What is Python?

Python is a programming language, released in 1991, written by Guido van Rossum.The language is based on the insights from professor Lambert Meertens, who wrote the language ABC, which is based on BASIC, but with all sorts of very sophisticated data structures. It's design philosophy is based around readability, notable using significant white spaces.

Python has gotten his name thanks to Guido's favorite television show: "Monty Python's Flying Circus".

History

Guido Van Rossum published the first version of Python code (version 0.9.0) at alt.sources in February 1991. This release included already exception handling, functions, and the core data types of list, dict, str and others. It was also object oriented and had a module system.

Python version 1.0 was released in January 1994. and 6 years later, 2.0 was released. this version ruled the world for 8 years, when in 2008 3.0 was released

All versions where backward compatible, except for the latest version.
